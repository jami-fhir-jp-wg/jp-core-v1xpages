Title

The Unified Code for Units of Measure (UCUM) is a code system intended to include all units of measures being contemporarily used in international science, engineering, and business.

This Code system is referenced in the content logical definition of the following value sets:

  • CommonUCUMCodesForAge
  • CommonUCUMCodesForDistance
  • CommonUCUMCodesForDuration
  • BodyLengthUnits
  • BodyTemperatureUnits
  • BodyWeightUnits
  • Common UCUM units
  • UCUMCodes
  • VitalSignsUnits
  • UnitsOfTime
  • Hl7VSWeightUnits
  • Hl7VSVolumeUnits
  • Hl7VSWeightUnits
  • Hl7VSVolumeUnits
  • Hl7VSTemperatureUnits
  • Hl7VSDonationDurationUnits
  • UnitsOfMeasureCaseSensitive

0. Range
Definition

範囲指定された時間の上限下限


低および高制限で定義された一連の順序付けされた数量。 / A set of ordered Quantities defined by a low and high limit.

Short範囲指定された時間の上限下限Low and Highに制限された値のセット / Set of values bounded by low and high
Comments

時間の上限量、下限量の範囲を持っている。単位指定された数量を割り当てている。Low,Highの値は時間の単位当てはめる。


記載されている低値と高い値は、範囲内にある値を決定する際に、任意に高い精度を持っていると想定されています。すなわち1.99は範囲2-> 3にありません。 / The stated low and high value are assumed to have arbitrarily high precision when it comes to determining which values are in the range. I.e. 1.99 is not in the range 2 -> 3.

rng-2: 存在する場合、低い値は高よりも低い値を持つものとします / If present, low SHALL have a lower value than high (low.empty() or high.empty() or (low <= high))

4. Range.low.value
Definition

時間範囲（下限）


測定量の値。値には、値の表示に暗黙の精度が含まれます。 / The value of the measured amount. The value includes an implicit precision in the presentation of the value.

Short時間範囲（下限）数値（暗黙の精度を持つ） / Numerical value (with implicit precision)
Comments

値の暗黙の精度は常に尊重されるべきです。金銭的価値には、精度を処理するための独自のルールがあります（標準的な会計の教科書を参照）。 / The implicit precision in the value should always be honored. Monetary values have their own rules for handling precision (refer to standard accounting text books).
